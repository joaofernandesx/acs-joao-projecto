Environmental Model
Request:  Information about a product/service the buyer is looking for.  Product/service description and the average amount of money it’s willing to spend.

Offer:  Information about a product/service offered by a seller. Product/service description and the basic market value defined by the seller.

PotentialSellers: List containing information about the sellers that provide a product/service that the buyer is looking for. The list is sorted with a connection between a request and its sellers. There is a different instance of this list for each buyer.

Role Model

Buyer

Description:

This role involves making Requests and negotiating.

Protocols and Activities:

PlaceRequest, NegotiateItem, RetractRequest, AddPotentialSeller, RemovePotentialSeller

Permissions:

changes PotentialSellers

Responsibilities:

Liveness:   ( PlaceRequest.NegotiateItem.RetractRequest )+

Safety:  TransactionValue <= MaximumPredefinedAmount for the request

 

Seller

Description:

This role involves making Offers and negotiating.

Protocols and Activities:

PlaceOffer, RetractOffer, NegotiateItem, UpdatePrices

Permissions:

reads  Offer

Responsibilities:

Liveness:   ( PlaceOffer.NegotiateItem.RetractOffer)+

Safety:  TransactionValue >= MinimumPredefinedPrice of the specific offer

 

Matchmaker

Description:

This role involves pairing up compatible Offers and Requests.

Protocols and Activities:

FindMatches, AddOffer, DelOffer, AddRequest, DelRequest, InformBuyer

Permissions:

changes Request, Offer

Interaction model

Protocol Name

PlaceRequest

 

Initiator

Buyer

Partner

Matchmaker

Input: Request Info

The buyer makes the request to the Matchmaker which places the information about the requested item in the requests database.

Output: RequestID



 

Protocol Name

PlaceOffer

 

Initiator

Seller

Partner

Matchmaker

Input: Product/service Info

The seller makes the offer to the Matchmaker which places the information about the product/service in the offer data base.

Output: OfferID



 

Protocol Name

RetractRequest

 

Initiator

Buyer

Partner

Matchmaker

Input: RequestID

The Matchmaker removes the information about the requested item from the requests data base.

Output



 

Protocol Name

RetractOffer

 

Initiator

Seller

Partner

Matchmaker

Input: OfferID

The Matchmaker removes the information about the product/service from the offer data base.

Output



 

 

Protocol Name

NegotiateItem

 

Initiator

Buyer

Partner

Seller

Input: Buyer Starting Price

The Buyer and the Seller negotiate the trade.

Output: Accept/Refuse

InformBuyer

 

Initiator

Matchmaker

Partner

Buyer
